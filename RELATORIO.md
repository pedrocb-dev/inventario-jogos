# RELATÓRIO

## Contexto e Planejamento

### Tema

Inventário de Jogos de Tabuleiro.

### Descrição

Aplicação web desenvolvida em Laravel para gerenciar um inventário de jogos de tabuleiro, permitindo cadastrar, editar, listar e excluir jogos após autenticação.

### Plano de Implementação

O projeto foi desenvolvido na seguinte ordem:

1. Configuração do Laravel.
2. Instalação do Laravel Boost.
3. Elaboração do Plano de Implementação.
4. Configuração da autenticação.
5. Criação das migrations, models e seeders.
6. Implementação do CRUD de Jogos.
7. Aplicação de uma identidade visual simples.
8. Documentação do projeto.

---

## Ferramentas de IA

### MCP utilizado

Foi utilizado o MCP configurado no Laravel Boost para auxiliar o desenvolvimento da aplicação.

### Skills desenvolvidas

Skills obrigatórias:

- Identidade Visual
- CRUD

Skills adicionais:

- Segurança
- Banco de Dados

---

## Desenvolvimento

Foram implementadas as seguintes funcionalidades:

- Login de usuários
- Cadastro de jogos
- Listagem de jogos
- Edição de jogos
- Exclusão de jogos
- Migrations
- Seeders
- Validação utilizando Form Requests

Durante o desenvolvimento foram realizados testes para validar o funcionamento da autenticação, banco de dados e CRUD.

---

## Conclusão

A aplicação atende aos requisitos mínimos propostos na atividade.

O uso do Laravel Boost e das Skills auxiliou na organização do desenvolvimento, enquanto a utilização da IA acelerou a implementação das funcionalidades.

Como limitação, o sistema possui apenas um CRUD, suficiente para atender aos objetivos da atividade.
